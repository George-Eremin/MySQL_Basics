База данных music необходима для работы предполагаемого сервиса онлайн-прослушивания музыки наподобие Яндекс.Музыки.

- Запросы на выборки сохранены в файле selects.sql

- Хранимая процедура генерит список нужного размера из id рекомендуемых треков для пользователя на основании жанров треков, сохраненных в плейлистах пользователя, или прослушанных пользователем.

- Триггер в таблице history выдает ошибку при попытке вставить время прослушивания в будущем.
